## Vscode Shortcuts

### Ctrl

- `ctrl + A`: 选中当前文件所有内容
- `ctrl + D`: 选中当前词
- `ctrl + F`: 查找
- `ctrl + G`: 跳转到指定行
- `ctrl + H`: 替换
- `ctrl + L`: 选中当前行
- `ctrl + N`: 新建窗口
- `ctrl + O`: 打开文件
- `ctrl + P`: 按名称搜索文件
- `ctrl + R`: 当前窗口切换到其他窗口
- `ctrl + S`: 保存
- `ctrl + V`: 粘贴
- `ctrl + W`: 关闭当前窗口
- `ctrl + F4`: 关闭当前窗口
- `ctrl + X`: 剪切 (选中的内容，或光标所在的一整行)
- `ctrl + Z`: 撤销
- `ctrl + /`: 注释当前行
