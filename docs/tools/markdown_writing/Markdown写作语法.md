# Markdown 写作语法

### 加粗

<strong>这段文字加粗</strong>

```markdown
<strong>这段文字加粗</strong>
**这段文字加粗**
```

### 斜体

<em>这段文字斜体</em>

```markdown
<em>这段文字斜体</em>
*这段文字斜体*
```

### 删除线

<del>这段文字删除线</del>

```markdown
<del>这段文字删除线</del>
~~这段文字删除线~~
```

### 下划线

<u>这段文字下划线</u>

```markdown
<u>这段文字下划线</u>
```

### 超链接

[这是一个超链接](https://www.baidu.com)

```markdown
[这是一个超链接](https://www.baidu.com)
```

### 图片

![这是一个图片](https://www.baidu.com/img/bd_logo1.png)

```markdown
![这是一个图片](https://www.baidu.com/img/bd_logo1.png)
```

### 换行

这是一段文字<br>这是一段文字

```markdown
这是一段文字<br>这是一段文字
```

### 居中

<div align='center'>

这是一段居中的文字
</div>

```markdown

<div align='center'>

这是一段居中的文字
</div>
```

### 代码块

```python
print('这是一个代码块')
```

````markdown
```python
print('这是一个代码块')
```
````
